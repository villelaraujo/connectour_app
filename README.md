# Connectour App

    App made for learning Nuxt js fullstack framework with Postgresql database.

    Simple app for creating Travel Logs and Logbooks, with login functionality.
    App is not deployed yet.

## Setup

    For the source code:
    *Create PostgreSQL database and name it: connectourApp
    *Run NPM INSTALL for node packages
    *Run NPX PRISMA MIGRATE to migrate the database schemas
    *Run NPM RUN DEV to initiate app on browser window
